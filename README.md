# fullstack-nonprofit-site

### A fullstack website for a fake non-profit organization.

#### Features:
- User login & registration authentication
- E-commerce system
- Payment Integration

#### Built with:
- **Frontend**
  - HTML
  - CSS
  - Javascript
  - EJS
- **Backend**
  - NodeJS
  - ExpressJS
  - PostgreSQL
